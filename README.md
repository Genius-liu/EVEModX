# EVEModX
EVEModX is a framework which helps you inject python code to EVE Online process(exefile.exe).

**WARNING: THIS IS A HACK OF EVE. ANY USE OF THIS CODE MAY BE PUNISHED BY THE OFFICIAL. USE AT YOUR OWN RISK.**
